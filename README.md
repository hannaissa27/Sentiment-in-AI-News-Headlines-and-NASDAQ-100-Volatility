A quantitative study of whether the tone of AI-related news headlines correlates with short-term NASDAQ-100 volatility, and how that differs between US and European outlets. Scoring headlines from the NYT and The Guardian against intraday windows, it tests for a weak but non-random link, stronger for US media due to home bias.

Code included was used as the study's data collection process to collect and analyze articles. Output is a CSV file with data.
